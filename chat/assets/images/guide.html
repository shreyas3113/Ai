AADI CLOUD User Guide

About AADI CLOUD

AADI CLOUD is an AI-powered chat application that allows you to interact with multiple AI models simultaneously. The "Collective Mind" feature combines responses from different models to provide comprehensive answers.

Getting Started

1. Starting a New Chat

Click the New Chat button in the sidebar to begin a fresh conversation.

2. Selecting AI Models

In the sidebar, use the AI Models section to select up to 3 different models to chat with simultaneously.

Key Features

Ensemble Mode

After receiving responses from individual models, click the Ensemble Mode button to see a synthesized response combining insights from all selected models.

Sidebar Navigation

Use the sidebar to access your chat history, manage AI models, and access settings.

Theme Toggle

Switch between light and dark modes using the theme toggle button in the header.

Troubleshooting

  • If responses don't appear, check your internet connection
  • Ensure you've selected at least one AI model
  • Try refreshing the page if features aren't working
